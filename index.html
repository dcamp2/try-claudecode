Claude Code 가이드

설치 · 커넥터 · 배포까지 — 클릭만 따라하면 끝!

시작해볼까요?
1

📦 Node.js 설치

Claude Code를 실행하려면 Node.js라는 프로그램이 필요해요.

⬇️ Node.js 다운로드
  • 다운로드된 파일을 더블클릭
  • "I accept" 체크하고 동의
  • 나머지는 계속 Next → Next → Install
  • 설치 끝나면 Finish 클릭
💡 왜 설치하나요? Claude Code가 Node.js 위에서 돌아가요. 엔진 같은 거예요!
2

🔧 Git 설치

코드 관리에 필요한 Git도 설치해요.

⬇️ Git 다운로드
  • 다운로드된 파일을 더블클릭
  • 아무것도 건드리지 말고 Next만 계속 클릭
  • 마지막에 Install → Finish
💡 왜 설치하나요? Claude Code가 파일을 관리할 때 Git을 사용해요. 비서의 필수 도구예요!
3

⚡ PowerShell 설정

Windows 보안 설정을 잠깐 바꿔야 해요.

  • 시작 버튼을 누르고 PowerShell 입력
  • "관리자 권한으로 실행" 클릭
  • "변경을 허용하시겠습니까?" → 예 클릭

아래 명령어를 복사해서 PowerShell에 붙여넣기 후 Enter!

Set-ExecutionPolicy RemoteSigned

"Y" 입력하고 Enter 누르면 끝!

💡 팁: 파란색 창이 뜨면 성공이에요. 겁먹지 마세요!
4

💻 Claude Code 설치

아까 열어둔 PowerShell에서 아래 명령어를 복사해서 붙여넣기 후 Enter!

npm install -g @anthropic-ai/claude-code

설치가 끝날 때까지 기다려주세요.

💡 팁: 글씨가 주르륵 올라가면 정상이에요. 끝나면 다시 입력할 수 있게 돼요!
5

🚀 Claude Code 실행

같은 PowerShell에서 아래 명령어를 하나씩 복사 → 붙여넣기 → Enter!

① 바탕화면에 test 폴더 만들고 이동하기

cd ~/Desktop; mkdir test; cd test

② Claude Code 실행하기

claude --dangerously-skip-permissions
💡 팁: 처음 실행하면 잠깐 로딩이 걸려요. 기다리면 돼요!
6

🔑 로그인하면 끝!

Claude Code가 실행되면 로그인 화면이 나와요.

  • 브라우저가 자동으로 열려요
  • Claude 계정으로 로그인
  • 돌아오면 바로 사용 가능! 🎉
💰 요금제 안내:
Pro 요금제 ($20/월)로도 쓸 수 있지만, 금방 사용량 제한에 걸려요.
넉넉하게 쓰려면 Max 요금제 ($100/월 또는 $200/월)를 추천해요!
7

⚡ cc 한 줄로 실행하기 (선택)

매번 claude --dangerously-skip-permissions 길게 치기 귀찮죠?
cc.bat 파일을 만들어두면 어느 터미널에서나 cc 한 글자로 실행할 수 있어요.

⬇️ cc.bat 다운로드
  • 다운로드된 cc.bat 파일을 찾아요
  • C:\Windows 폴더에 복사해서 붙여넣기
  • "관리자 권한이 필요합니다" → 계속 클릭
  • 이제 어느 터미널·PowerShell에서나 cc 입력 → Enter!
💡 원리: C:\Windows는 PATH에 등록된 폴더라서, 거기에 있는 .bat 파일은 어디서든 이름만으로 실행할 수 있어요. cc.bat 안에는 그냥 claude --dangerously-skip-permissions가 적혀있을 뿐이에요!
🎉

설치 완료!

이제 Claude Code에게 뭐든 시켜보세요.
"홈페이지 만들어줘" 라고 말해보는 건 어때요?

💎 BONUS

한 걸음 더 — 내 비서로 키우기

Claude 데스크톱을 깔고 Gmail · 캘린더를 연결하면
Claude Code가 내 이메일과 일정까지 보고 일을 처리해줘요.

1

🖥️ Claude 데스크톱 설치

Windows용 Claude 데스크톱 앱이 필요해요. 커넥터(Gmail·캘린더 같은 외부 연결)를 여기서 켜요.

⬇️ Claude 데스크톱 다운로드
  • 다운로드된 파일을 더블클릭해서 설치
  • 설치 끝나면 Claude 앱 실행
  • 아까와 같은 Claude 계정으로 로그인
💡 잠깐: 이건 Claude Code(터미널)랑 다른 별도 앱이에요. 같은 계정으로 로그인하면 데이터가 연동돼요!
2

🔌 Gmail · 캘린더 커넥터 연결

Claude 데스크톱 안에서 커넥터(Connectors)를 켜면 Claude가 내 이메일·일정을 직접 볼 수 있어요.

  • Claude 데스크톱 좌측 하단의 내 이름/이니셜 클릭
  • Settings (설정) → Connectors (커넥터) 메뉴로 이동
  • Gmail 옆 Connect 버튼 클릭
  • 브라우저가 열리면 Google 계정으로 로그인 → 권한 허용
  • 같은 방법으로 Google Calendar도 연결
🔒 안심하세요: 내가 시킬 때만 Claude가 데이터를 봐요. 언제든 같은 메뉴에서 연결을 끊을 수 있어요.
3

✨ 이제 Claude Code에 시키기

한 번 연결해두면 Claude Code 터미널에서도 같은 커넥터를 쓸 수 있어요.
아까 만든 cc 명령으로 켜고 이렇게 말해보세요.

내 이메일과 캘린더 확인해서
오늘 내 업무 요약보고서 만들어줘
💡 이런 것도 됩니다:
· "이번 주 회의 일정 정리해서 핵심만 알려줘"
· "지난 3일간 받은 중요 메일 5개 요약해줘"
· "오후에 비는 시간 찾아서 운동 일정 잡아줘"
· "프로젝트 X 관련 메일·일정 묶어서 보고서로 만들어줘"
🚀

비서 준비 완료!

Claude가 내 메일과 일정을 보고
일까지 처리해주는 든든한 비서가 됐어요.

🌐 DEPLOY

내가 만든 사이트, 세상에 공개하기

Claude Code로 홈페이지나 도구를 만들었나요?
무료로 인터넷에 띄워서 친구·동료에게 보여줄 수 있어요.

1

🐙 GitHub 계정 만들기

GitHub은 코드·파일을 저장하는 온라인 창고예요. 무료고, 한 번 만들면 평생 써요.

⬇️ GitHub 가입하기
  • 이메일·비밀번호 입력 (로봇 검증 한 번 통과)
  • 유저네임 정하기 — 영어/숫자로 (예: joon-dev)
  • 이메일로 온 인증 코드 입력 → 가입 완료
  • 플랜 선택 화면 → "Continue for free" 선택
💡 유저네임은 신중하게: 나중에 사이트 주소(유저네임.github.io)에도 쓰여요. 짧고 깔끔한 걸로!
2

📤 Claude Code로 GitHub에 올리기

가장 쉬운 방법 — Claude Code에게 시키는 거예요.

아까 만든 cc로 Claude Code 켜고 이렇게 말하세요.

이 프로젝트를 내 GitHub에 올려줘.
내 GitHub 유저네임은 ○○○이야.

Claude가 알아서 git 명령어를 실행하고 GitHub 저장소(repository)를 만들어줘요.

  • 중간에 GitHub 로그인을 요청할 수 있어요 → 브라우저에서 인증
  • 저장소 이름은 Claude가 추천해주거나, 원하는 이름을 알려줘도 돼요
  • 끝나면 github.com/유저네임/저장소이름 주소를 알려줘요
💡 손으로 하고 싶다면: github.com에서 "New repository" 만들고 파일을 드래그 앤 드롭해도 돼요. 하지만 Claude한테 시키는 게 훨씬 빨라요!
3

▲ Vercel 가입

Vercel은 웹사이트를 인터넷에 띄워주는 무료 서비스예요.

⬇️ Vercel 가입하기
  • "Continue with GitHub" 클릭 (방금 만든 계정으로)
  • 권한 묻는 화면 → "Authorize Vercel" 클릭
  • 플랜 선택 → 무료 "Hobby" 선택
  • 이름·소속 같은 건 가볍게 채우고 통과
4

✨ 클릭 한 번으로 배포

Vercel 대시보드에서 GitHub 저장소만 골라주면 끝이에요.

  • 대시보드 우측 상단 "Add New..." → "Project" 클릭
  • 아까 만든 저장소 옆 "Import" 클릭 (안 보이면 "Adjust GitHub App Permissions"로 권한 추가)
  • 설정 화면이 떠도 아무것도 건드리지 말고 아래 "Deploy" 클릭
  • 30초~1분 기다리면 🎉 → "Visit" 클릭하면 새 사이트 오픈!
🌐 결과: ○○○.vercel.app 같은 주소가 생기고, 이제 전 세계 누구나 그 URL로 접속할 수 있어요!
💡 자동 업데이트: 이후 Claude Code로 사이트 수정하고 GitHub에 다시 push하면 Vercel이 자동으로 재배포해요. 손댈 거 없음!
🌐
