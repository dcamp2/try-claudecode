Claude Code 설치 가이드

Windows에서 6단계만 따라하면 끝!

시작해볼까요?
1

📦 Node.js 설치

Claude Code를 실행하려면 Node.js라는 프로그램이 필요해요.

⬇️ Node.js 다운로드
  • 다운로드된 파일을 더블클릭
  • "I accept" 체크하고 동의
  • 나머지는 계속 Next → Next → Install
  • 설치 끝나면 Finish 클릭
💡 왜 설치하나요? Claude Code가 Node.js 위에서 돌아가요. 엔진 같은 거예요!
2

🔧 Git 설치

코드 관리에 필요한 Git도 설치해요.

⬇️ Git 다운로드
  • 다운로드된 파일을 더블클릭
  • 아무것도 건드리지 말고 Next만 계속 클릭
  • 마지막에 Install → Finish
💡 왜 설치하나요? Claude Code가 파일을 관리할 때 Git을 사용해요. 비서의 필수 도구예요!
3

⚡ PowerShell 설정

Windows 보안 설정을 잠깐 바꿔야 해요.

  • 시작 버튼을 누르고 PowerShell 입력
  • "관리자 권한으로 실행" 클릭
  • "변경을 허용하시겠습니까?" → 예 클릭

아래 명령어를 복사해서 PowerShell에 붙여넣기 후 Enter!

Set-ExecutionPolicy RemoteSigned

"Y" 입력하고 Enter 누르면 끝!

💡 팁: 파란색 창이 뜨면 성공이에요. 겁먹지 마세요!
4

💻 Claude Code 설치

아까 열어둔 PowerShell에서 아래 명령어를 복사해서 붙여넣기 후 Enter!

npm install -g @anthropic-ai/claude-code

설치가 끝날 때까지 기다려주세요.

💡 팁: 글씨가 주르륵 올라가면 정상이에요. 끝나면 다시 입력할 수 있게 돼요!
5

🚀 Claude Code 실행

같은 PowerShell에서 아래 명령어를 하나씩 복사 → 붙여넣기 → Enter!

① 바탕화면에 test 폴더 만들고 이동하기

cd ~/Desktop; mkdir test; cd test

② Claude Code 실행하기

claude --dangerously-skip-permissions
💡 팁: 처음 실행하면 잠깐 로딩이 걸려요. 기다리면 돼요!
6

🔑 로그인하면 끝!

Claude Code가 실행되면 로그인 화면이 나와요.

  • 브라우저가 자동으로 열려요
  • Claude 계정으로 로그인
  • 돌아오면 바로 사용 가능! 🎉
